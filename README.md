# oopsie - privacy policy


Last updated: March 30, 2026

Oopsie does not collect, transmit, or share any personal data. All replacement rules and settings are stored locally on your device using Chrome's built-in storage API (chrome.storage.local) and never leave your browser.

Oopsie does not use analytics, tracking, cookies, or external servers.
